# UniversityNET


前言
-
在中山大学南方学院的逼迫之下被迫开始写shell，稍微保存一下路由器用来维持网络稳定性的小脚本。因为学校一个账户可以同时登陆电脑和手机端的特性，使网络速度最高可以达到原来的两倍。请勿用作断网等其他恶意行为(虽然也不是不行)。:smiling_imp:	
___
环境要求
-
linux  
curl
