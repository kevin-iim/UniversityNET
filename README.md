# UniversityNET


前言
-
在中山大学南方学院的逼迫之下被迫开始写shell，稍微保存一下路由器用来维持网络稳定性的小脚本。因为学校一个账户可以同时登陆电脑和手机端的特性，使每个宿舍的最高网络速度可以达到上行15M|下行60M(联通)。请勿用作断网等其他恶意行为(虽然也不是不行)。:smiling_imp:	
___
已知最低环境要求
-
linux(openwrt)  
curl 7.60.0 (arm-openwrt-linux-gnu) libcurl/7.60.0 mbedTLS/2.14.1
